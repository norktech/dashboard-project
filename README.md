# 📈 Data Dashboard — NorkTech

Automatically generates a bar chart visualizing book prices collected by the web scraper.

## 🛠️ Technologies
- Python
- Matplotlib
- OpenPyXL

## ▶️ How to run
pip install matplotlib openpyxl
python dashboard.py

## 📊 Output
Generates dashboard.png with a bar chart of all book prices.
